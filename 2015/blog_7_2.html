Transform属性应用于元素的2D或3D转换。这个属性允许你将元素旋转，缩放，移动，倾斜等。 #### 其次、CSS3 transform都有哪些常用属性？ transform的属性包括：rotate()[旋转] / skew()[倾斜] / scale()[缩放] / translate()[位移] ，分别还有x、y之分，比如：rotatex() 和 rotatey() ，以此类推。 **注：效果的实现需要两种属性的支撑：A、过渡效果(transition); B、关键帧动画(animation)** transition和animation的区别： transition需要触发一个事件 ,而animation在不需要触发任何事件的情况下也可以显式的随着时间变化来改变元 素css的属性值，从而达到一种动画的效果。 一、rotate [尝试一下》](file:///F:/transform/rotate.html) 语法(关键帧): ``` @-webkit-keyframes mymove { 0/from{-webkit-transform: rotate(90deg)}; 50%{-webkit-transform: rotate(180deg)}; 100%/to{-webkit-transform: rotate(360deg)}; } ``` ``` #myDIV { width: 200px; height:200px; background-color: coral; -webkit-animation: mymove 5s infinite; /* Chrome, Safari, Opera */ } ```
