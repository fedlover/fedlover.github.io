## 数组常用的11个方法 首先明确，我们需要从四个维度总结数组的常用方法: 1. 此方法的作用什么 2. 此方法有无参数；如果有，参数是什么 3. 此方法是否有返回值；如果有，返回值是什么 4. 原数组是否发生变化 ### push 1. 将数组末尾追加一项 2. 参数是要增加那一项 3. 返回值是追加一项后新数组的长度 4. 原数组发生变化 ``` var arr=[11,22,33]; var newArr=arr.push(44); console.log(newArr,arr); // newArr->4 arr->[11,22,33,44] ``` ### pop 1. 删除数组末尾一项 2. 没有参数 3. 返回值是删除的那一项（也就是数组最后一项） 4. 原数组发生变化 ``` var arr=[11,22,33]; var newArr=arr.pop(); console.log(newArr,arr); // newArr->33 arr->[11,22] ``` ### unshift 1. 向数组首位追加一项 2. 参数是追加的内容 3. 返回值是新数组的长度 4. 原数组发生变化 ``` var arr=[11,22]; var newArr=arr.unshift(33); console.log(newArr,arr); // newArr->4 arr->[33,11,22] ``` ### shift 1. 删除数组第一项 2. 没有参数 3. 返回值是删除的内容 4. 原数组发生变化 ``` var arr=[11,22,33]; var newArr=arr.shift(); console.log(newArr,arr); // newArr->4 arr->[22,33] ``` ### splice splice的用法很多 1. 参数有三个： no1：截取的开始索引；no2：截取的个数；no3：往数组中添加的内容； 2. 根据参数的不同，可以截取数组中的某一项，或者连续几项，或者往数组某个位置中添加项，返回值都是截取的到的项 3. 返回值是截取的内容 4. 原数组都发生变化 *** - 只有一个参数，表示一直截取到数组最后 ``` var arr=[1,2,3,4,5,6]; var newArr=arr.splice(4); // 截取索引为4到最后的所有项 console.log(newArr,arr); // newArr->[5,6] arr->[1,2,3,4] ``` - 两个参数，可以截取若干项 ``` var arr=[1,2,3,4,5,6]; var newArr=arr.splice(1,3); //从索引1开始，截取3个 console.log(newArr,arr); //newArr->[2,3,4] arr->[1,5,6] ``` - 三个参数，可以替换 ``` var arr=[1,2,3,4,5,6]; var newArr=arr.splice(1,3,2,2,2); //从索引1开始，截取3个，并替换成2,2,2 console.log(newArr.arr); //newArr->[2,3,4] arr->[1,2,2,2,5,6] ``` - 如果想直接给数组添加项，可以使用 `arr.splice(n,0,n,n,n)` 可以直接添加到索引n处的前面 - `arr.splice(0,0,111)` 就相当于在数组首位添加内容，效果和 `arr.unshift(111)` 是一样的 ### reverse 1. 反转数组 2. 没有参数 3. 返回值是反转后的新数组 4. 原数组发生变化 ``` var arr=[1,2,3,4,5,6]; var newArr=arr.reverse(); console.log(newArr.arr); //newArr->[6,5,4,3,2,1] arr->[6,5,4,3,2,1] ``` ### concat 1. 合并数组 2. 参数是需要和原数组进行合并的数组 3. 返回值是合并后的新数组 4. 原数组没有发生变化 ``` var arr=[1,2,3]; var array=[4,4,4]; var newArr=arr.concat(array); console.log(newArr,arr); // newArr->[1,2,3,4,4,4] arr->[1,2,3] ``` ### join 1. 将数组转为字符串 2. 没有参数 3. 返回值是转变的字符串 4. 原数组没有变化 ``` var arr=[1,2,3]; var newStr=arr.join(); console.log(newStr,arr); // newStr->'1,2,3' arr->[1,2,3] ``` ### slice 1. 参数有三个； no1：开始的索引；no2：结束的索引；（参数支持负数，表示从末尾开始算起） 2. 根据参数的不同，可以复制数组一部分，或者一项，或者复制整个数组 3. 返回值是复制出来的新数组 4. 原数组没有发生变化 *** - 一个参数,可以复制某一索引以后的所有项；如果这个参数为0，就是复制整个数组 ``` var arr=[1,2,3,4,5,6]; var newArr=arr.slice(3); //复制索引 3 以后所有项 console.log(newArr.arr); // newArr->[4,5,6] arr->[1,2,3,4,5,6] ``` - 两个参数，复制指定的若干项（包前不包后） ``` var arr=[1,2,3,4,5,6]; var newArr=arr.slice(0,2); // 复制从索引为0到索引为 2 的项，包括第 0 项，不包括第 2 项 console.log(newArr,arr); // newArr->[1,2] ``` ### sort 1. 将数组从小到大，或者从大到小排序 2. 参数是一个回调函数 3. 返回值 4. 原数组没有发生变化 *** - 数字，字母可以直接用 ``` var arr=[2,5,8,9,0]; var newArr=arr.sort(funcion(a,b){return a-b;}); // return b-a 可以从大到小排列 console.log(newArr,arr); // newArr->[0,2,5,8,9] arr->[2,5,8,9,0] ``` - 数组的每一项是汉字，可以根据首字母，按照字母表顺序排列 ``` var arr=['我是','例子','不用','理我']; var newArr=arr.sort(function(a,b){return a.localeCompare(b))}); console.log(newArr,arr); //newArr->["不用","理我","例子","我是"] arr->['我是','例子','不用','理我'] ``` *附加：* localeCompare()是用本地特定的顺序来比较两个字符串，返回值是说明比较结果的数字，如果参数小于调用这个方法的变量字符串，则localeCompare()返回小于 0 的数。反之，则该方法返回大于 0 的数。如果两个字符串相等，或根据本地排序规则没有区别，该方法返回 0。 ### forEach 1. 循环遍历数组 2. 参数是回调函数 3. 没有返回值 4. 原数组没有发生变化 ``` var arr=[11,22,33,44]; var newArr=arr.forEach(function(item,index,arr){ // item为循环的每一项，index是循环到的索引，arr为原数组 // 处理逻辑 }) ```
